# 정수기 경쟁사 포지셔닝맵

국내 6개 브랜드(코웨이·LG·청호나이스·쿠쿠·SK매직·교원웰스) 렌탈 정수기 **183종**을
**출시시기(가로축) × 5년(60개월) 총 렌탈료(세로축)** 기준으로 배치한 인터랙티브 포지셔닝맵입니다.

## 기능
- 제품 사진에 **마우스를 올리거나 클릭**하면 상세 카드(제품명·모델명·출시월·월 렌탈료·5년 총액·일시불가)가 표시됩니다.
- 상단 범례에서 **브랜드를 클릭하면 그 브랜드만** 보이고, 사진이 자동으로 커집니다. 다시 누르거나 **"전체"**를 누르면 모두 보기로 돌아갑니다.
- 출시시기 미확인 12종은 좌표에 배치할 수 없어 맵 아래 **별도 트레이**에 모아두었습니다.

## 바로 보기 (GitHub Pages)
저장소에 올린 뒤 아래 순서로 배포하면 웹 주소로 바로 열람할 수 있습니다.

1. GitHub에서 새 저장소를 만들고 이 폴더의 파일(`index.html`, `README.md`)을 업로드합니다.
2. 저장소 상단 **Settings → Pages** 이동
3. **Build and deployment > Source** 를 `Deploy from a branch` 로 두고, Branch 를 `main` / 폴더 `/ (root)` 로 지정 후 **Save**
4. 1~2분 뒤 상단에 표시되는 주소(`https://<사용자명>.github.io/<저장소명>/`)로 접속하면 됩니다.

> 파일이 단일 HTML(사진 포함 약 610KB)이라 별도 빌드 과정이나 서버가 필요 없습니다. 그냥 `index.html`을 브라우저로 열어도 동일하게 작동합니다.

## 데이터 기준
- **가격**: 브랜드별 기본 약정개월(36/60/72/84개월 등)이 달라, 6개 브랜드 공통인 **5년(60개월) 약정** 기준 총 렌탈료로 통일했습니다.
- **산정근거 표시**
  - `실측` — 원본에 5년 약정 공식요금이 존재(LG, SK매직, 교원웰스 일부).
  - `추정` — 월 렌탈료가 약정과 무관하게 고정이라고 가정하고 ×60 환산한 참고치(코웨이·청호나이스·쿠쿠 등).
- **출시시기**: 브랜드별 원본 표기를 `YYYY-MM`으로 통일했으며, 원본에 없던 값은 제조사 공식몰·보도자료·다나와 등에서 보완했습니다. 12종은 확인되지 않아 미배치 상태입니다.
